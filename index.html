PocketMine-MP is a server software for Minecraft PE (Pocket Edition) done in PHP. It has a Plugin API that enables a developer to extend it and add new features, or change default ones

The entire server is done in PHP, and has been tested, profiled and optimized to run smoothly.

Check the sidebar for the latest Stable or Development release.
