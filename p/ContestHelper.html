ContestHelper helps you hold Contest events.

Commands

<> is an essential param.
[] is an optional param.

setarea <PlayerName>

The command which set the present coordinates of <PlayerName> as his point area.
When someone place one block there, the player assigned the area gets one point.
When someone break one block there, the player assigned the area loses one point.

ranking [limit]

The command which display the ranking up to [limit] place.
default : [limit] = 3.
