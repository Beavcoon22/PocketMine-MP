WorldEditor is a port of WorldEdit to PocketMine

Tested in Alpha_1.0.2

  • Download 0.3

WorldEditor is a port of WorldEdit to PocketMine so you can easily modify your worlds in-game, using the server console and a player as a pointer.

You can modify the default-player option in the plugin config to change the default target player.

Commands

  • //help: Shows WorldEditor help
  • //pos1 [player]: Choose the first position using the default player or the given one
  • //pos2 [player]: Choose the second position using the default player or the given one
  • //set [block]: Sets the entire selection to [block]
  • //replace [target] [block]: Replace [target] with [block] in the entire selection. If not metadata is given, all blocks of that ID will be targeted
